# 兴趣记录 · 热力图

网页版兴趣记录工具，用类似 GitHub 贡献热力图的方式记录并展示你的兴趣打卡次数。

## 功能

- **热力图展示**：以 53 周 × 7 天的格子热力图展示每日记录次数，颜色越深表示次数越多
- **热力图命名**：每个热力图可单独命名（如「阅读」「运动」）
- **多热力图**：支持创建无限个兴趣热力图，左侧列表切换
- **自定义颜色**：每个热力图可单独选择主题色，热力深浅随该颜色自动渐变
- **点击记录**：点击某天格子即可增加当日记录次数
- **本地持久化**：数据保存在浏览器 localStorage，刷新不丢失

## 使用方式

用浏览器直接打开 `index.html` 即可使用，无需安装或服务器。

```bash
# 可选：用本地服务器打开（避免部分环境对 file:// 的限制）
npx serve .
# 然后访问 http://localhost:3000
```

## 文件说明

- `index.html`：页面结构
- `styles.css`：样式（深色主题、热力图布局）
- `app.js`：逻辑（热力图生成、多图管理、颜色、本地存储）

## 技术说明

- 纯前端实现，无后端
- 数据键：`habits-tracker-heatmaps`（localStorage）
- 热力图时间范围：最近约 53 周（约一年），以当前周为最右一列
